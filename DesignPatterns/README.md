# 🏗️ Explorando Padrões de Projeto na Prática com Java

Este projeto tem como objetivo demonstrar a implementação de **Padrões de Projeto** em Java, reforçando boas práticas no desenvolvimento de software. Os padrões explorados neste projeto incluem **Singleton, Strategy e Facade**.

## 🎯 Padrões de Projeto Implementados

✅ **Singleton** → Garante que uma classe tenha apenas uma instância e fornece um ponto de acesso global a ela.  
✅ **Strategy** → Define uma família de algoritmos, encapsula cada um deles e os torna intercambiáveis.  
✅ **Facade** → Fornece uma interface simplificada para um conjunto de subsistemas mais complexos.  

## 🏗️ Estrutura do Projeto

O projeto segue a seguinte organização:

- **Singleton** → Implementação de um gerenciador de configurações com instância única.  
- **Strategy** → Diferentes estratégias de cálculo aplicadas dinamicamente.  
- **Facade** → Interface única para interagir com múltiplas classes do sistema.  

## 📌 Como Executar o Projeto

### Pré-requisitos
- Java 11+ instalado  
- IDE como IntelliJ, Eclipse ou VS Code (opcional)  

### Passos

1. Clone o repositório:
   ```sh
   git clone https://github.com/seu-usuario/design-patterns-java.git
   ```
2. Acesse o diretório do projeto:
   ```sh
   cd design-patterns-java
   ```
3. Compile o código:
   ```sh
   javac Main.java
   ```
4. Execute o programa:
   ```sh
   java Main
   ```
